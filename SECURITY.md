# Security Policy

## Supported Versions

This project is under active development. Security fixes apply only to the `main` branch.

## Reporting a Vulnerability

Please report suspected vulnerabilities **privately** via email to **borabkrr@gmail.com** with the subject line `[security] deha-crm`.

Include:
- A description of the issue and the affected component
- Steps to reproduce (proof-of-concept code or commit SHA if applicable)
- Your assessment of impact and severity

You will receive an acknowledgement within 72 hours. Public disclosure happens only after a fix has shipped, coordinated with the reporter.

Please do **not** open public GitHub issues for security reports.
