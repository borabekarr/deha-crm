# src/components/ui/

Shadcn UI primitives only (Button, Input, Dialog, etc.) — generated or installed via `pnpm dlx shadcn`.
These are unstyled, low-level building blocks with no business logic.
Do NOT put feature-specific components here. Do NOT put shared layout components here.
Anything with CRM domain knowledge belongs in `src/components/shared/` or a feature folder.
